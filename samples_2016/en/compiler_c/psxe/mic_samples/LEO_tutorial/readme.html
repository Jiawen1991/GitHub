LEO_tutorial
Overview
This tutorial illustrates the introductory level use of a system with the Intel® Xeon Phi™ coprocessor to execute an application on both the host CPU and the coprocessor. It illustrates how to compile the source code into an application that executes on both the host and target, and define code sections that execute only on the host.
System Requirements
Minimum requirements include an Intel® 64 architecture processor-based host system with an Intel® Xeon Phi™ coprocessor, a supported version of the Linux* operating system, and the Intel® Manycore Platform Software Stack (Intel® MPSS). Refer to the Intel® Parallel Studio XE product Release Notes for details on the complete system requirements.
Build Instructions
Expand the TAR .gz file to a writable directory of your choice.  Refer to the product documentation tutorials collection for the Tutorial: Using the Intel® Xeon Phi™ Coprocessor guide containing details on building and running this tutorial.
